# Backup & Restore Runbook

## Scripts

| Platform | Backup | Restore |
|---|---|---|
| Linux/macOS | `scripts/backup/pg-dump.sh` | `scripts/backup/pg-restore.sh` |
| Windows | `scripts/backup/pg-dump.ps1` | `scripts/backup/pg-restore.ps1` |

## Backup

```powershell
$env:POSTGRES_DB="society_connect"
$env:POSTGRES_USER="society"
$env:POSTGRES_PASSWORD="society"
$env:POSTGRES_HOST="localhost"
$env:POSTGRES_PORT="5432"
powershell -ExecutionPolicy Bypass -File scripts/backup/pg-dump.ps1
```

Output: `backups/society_connect-YYYYMMDD-HHMMSS.sql`

## Restore drill

```powershell
$env:RESTORE_DATABASE_URL="postgresql://society:society@localhost:5432/society_connect_restore"
powershell -ExecutionPolicy Bypass -File scripts/backup/pg-restore.ps1 -BackupFile backups/society_connect-latest.sql
```

Verify:

```sql
SELECT COUNT(*) AS societies FROM "Society";
SELECT COUNT(*) AS users FROM "User";
```
